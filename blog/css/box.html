css盒子模型

October 25, 2018 · 2 min read

概念​

CSS 盒模型本质上是一个盒子，盒子包裹着HTML 元素，盒子主要由四个属性组成，从内到外分别是：content 、padding 、border、margin，

我们可以直接看下面这张图，最外面橙色的就是外边距区域 margin，往里黄色的是边框区域 border，再往里的绿色的是内边距区域 padding，最里面绿色的就是内容区域 content 。

分类​

1. W3C标准盒模型​

整个元素的实际宽度 = border-left-width + padding-left + content-width + padding-right + border-right-width
整个元素的实际高度 = border-top-width + padding-top + content-height + padding-bottom + border-bottom-width

对应css：

box-sizing: content-box;

参考：

我们可以看到，虽然设置了 box1 的宽度为 200px，实际它的宽度是 280px，也就是 width + 左右 padding + 左右 border，实际高度是 280px，也就是 height + 上下 padding + 上下 border

2. IE怪异盒模型​

整个元素的实际宽度 = 实际设置的 width，width 包含了 左右内边距以及左右边框 整个元素的实际高度 = 实际设置的 height，height 包含了 上下内边距以及上下边框

对应css：

box-sizing: border-box;

参考：

最佳实践​

一般为了布局上的方便，我们将所有的元素都设置为 box-sizing: border-box;

html {
  box-sizing: border-box;
}

*,
*::before,
*::after {
  box-sizing: inherit;
}
